GitHub Token 設定

由於 GitHub 公開 API 每小時有 60 次存取限制，若您發現網頁無法載入（回傳 403 限制），請在此設定您的 Personal Access Token (PAT)。此 Token 將僅保存在您本地瀏覽器的 LocalStorage 中，安全可靠。
